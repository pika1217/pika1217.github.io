纯CSS实现自适应正方形

发表于 2017-08-04 | 分类于 CSS

自适应正方形是指宽度和高度随着屏幕宽度（不管是移动端还是PC端）进行等比例变化的正方形。

CSS3 vw 单位

CSS3 中新增了一组相对于可视区域百分比的长度单位 vw, vh, vmin, vmax。其中vw是相对于视口宽度百分比的单位，1vw = 1% viewport width，vh是相对于视口高度百分比的单位，1vh = 1% viewport height；vmin是相对当前视口宽高中较小的一个的百分比单位，同理vmax是相对当前视口宽高中 较大 的一个的百分比单位。

优点：简洁方便

缺点：浏览器兼容不好

设置垂直方向的padding百分比数值撑开容器

margin、padding的百分比数值是相对父元素宽度计算的，只需在垂直方向设置一个与width百分比相同的数值就可以出现自适应正方形。

注意：如果正方形中有内容，则会将高度拉伸content的宽度，此时我们可以设置height:0;此时的问题是在元素上设置max-height属性会失效，因为max-height属性只会对content height起作用。

利用伪元素的margin(padding)-top撑开容器

为了解决max-height属性设置问题,用一个子元素撑开 content 部分的高度，从而使 max-height 属性生效,如果要使用margin-top，要注意边界塌陷，在父元素中使用overflow:hidden
